# Salon Demo Export Notes

Build the Hair Salon homepage as a Gutenberg page using the included Webba patterns.

- Export `content.xml` from Tools > Export after creating the page, menu, and any supporting content.
- Export `widgets.wie` using a widget import/export plugin compatible with One Click Demo Import.
- Export `customizer.dat` using the One Click Demo Import Customizer Export/Import workflow.
- Create `preview.jpg` from a real screenshot of the finished demo.
